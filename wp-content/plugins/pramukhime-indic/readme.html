About PramukhIME Indic Wordpress Plugin

What is PramukhIME Indic Wordpress Plugin?

PramukhIME Indic Wordpress Plugin is a Wordpress plugin using which any Wordpress blogger can enable typing in Indian language without downloading/installing any special software.

About Developer

Gettting inspiration and blessings from H.D.H. Pramukh Swami Maharaj, Vishal Monpara has dedicated more than 3500 hours from January 2005 till January 2017 to create free utilities for typing in Indian languages.

Supported Wordpress Version

Wordpress 3.9 onwards is supported

Version

This plugin as a whole is marked as version 4.0.0. This plugin uses PramukhIME TinyMCE Plugin 4.0.0 which in turn uses uses PramukhIME Javascript Library 4.0.0.

Installation

  • Unzip the file on your local hard disk.
  • Upload all files to BLOGROOT/wp-content/plugins directory.
  • Activate via WordPress Plugins tab.
  • Go to Settings Menu > "PramukhIME" for setting your preference
